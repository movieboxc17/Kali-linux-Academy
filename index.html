Nmap (Network Mapper)

Nmap ("Network Mapper") is a free and open source utility for network discovery and security auditing. It uses raw IP packets in novel ways to determine what hosts are available on the network, what services those hosts are offering, what operating systems they are running, and dozens of other characteristics.

Basic Usage

nmap 192.168.1.1

Scans a single IP address

nmap 192.168.1.0/24

Scans an entire subnet

nmap -p 1-100 192.168.1.1

Scans ports 1-100 on the target

Advanced Options

nmap -sV -sC -O -p- 192.168.1.1

Performs a comprehensive scan with service detection (-sV), default scripts (-sC), OS detection (-O), and scans all ports (-p-)

Essential Kali Linux Commands

Master the command line with these essential Kali Linux commands.

File System Navigation

ls

List directory contents

cd [directory]

Change directory

pwd

Print working directory

find / -name [filename]

Find files in the system

File Manipulation

cat [file]

Display file contents

grep [pattern] [file]

Search for patterns in files

cp [source] [destination]

Copy files or directories

mv [source] [destination]

Move or rename files

Network Commands

ifconfig

Display network interfaces

ping [host]

Test network connectivity

netstat -tuln

Display network connections

traceroute [host]

Trace route to host

System Information

uname -a

Display system information

top

Display running processes

df -h

Show disk usage

free -m

Show memory usage

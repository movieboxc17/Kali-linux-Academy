Nmap (Network Mapper)

Reconnaissance Network Scanning Enumeration

Nmap ("Network Mapper") is a free and open source utility for network discovery and security auditing. It uses raw IP packets in novel ways to determine what hosts are available on the network, what services those hosts are offering, what operating systems they are running, and dozens of other characteristics.

Nmap is often used as the first step in the penetration testing process to discover network information before planning more targeted attacks.

Basic Usage

Scan a single IP address 
nmap 192.168.1.1
Scan an entire subnet 
nmap 192.168.1.0/24
Scan specific ports 
nmap -p 1-100 192.168.1.1

Advanced Options

Comprehensive scan with service detection, scripts and OS detection 
nmap -sV -sC -O -p- 192.168.1.1
Aggressive scan with all advanced options 
nmap -A 192.168.1.1

Output Options

Save results to files in all formats 
nmap -oA scan_results 192.168.1.1

Tips for Effective Usage

  • Use -sS for stealthy SYN scans that don't complete TCP connections
  • Add --reason to see why Nmap made certain determinations
  • Use -T options (0-5) to adjust timing, with higher numbers being faster but potentially less stealthy
  • Add --script vuln to scan for known vulnerabilities
  • Use -Pn if you know hosts are up but they're not responding to ping probes

Practice Exercise

Try to scan your local network to identify all active hosts, their operating systems, and open ports:

Full network scan 
sudo nmap -sS -O -sV 192.168.1.0/24 -oN network_scan.txt

Remember: Only scan networks you have permission to scan!

Quick Reference

Option Description
-sS SYN scan
-sV Version detection
-O OS detection
-p Port specification
-A Aggressive scan

Essential Kali Linux Commands

Master the command line with these essential Kali Linux commands.

File System Navigation

List directory contents
ls -la

Lists all files (including hidden ones) in detailed format

Change directory
cd /path/to/directory

Navigate to the specified directory

Print working directory
pwd

Display current directory path

Find files
find / -name "example.txt" 2>/dev/null

Find a file named "example.txt" in the entire system, hiding error messages

Network Commands

Display network interfaces
ifconfig

Shows all network interfaces and their configurations

Test network connectivity
ping -c 4 8.8.8.8

Sends 4 ping packets to Google's DNS server

Display network connections
netstat -tuln

Shows all active TCP/UDP listening ports

DNS lookup
dig example.com

Performs DNS lookup for the specified domain

System Information

Display system information
uname -a

Displays kernel version and system architecture

Process monitoring
htop

Interactive process viewer with more features than top

Disk usage
df -h

Shows disk space usage in human-readable format

Memory usage
free -m

Displays memory usage in megabytes

Users & Permissions

Change file permissions
chmod 755 filename

Sets read, write, execute for owner; read, execute for others

Change file ownership
chown user:group filename

Changes the owner and group of a file

Run command as root
sudo command

Executes the command with superuser privileges

List all users
cat /etc/passwd

Displays all user accounts on the system
